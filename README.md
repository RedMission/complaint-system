# 二维码投诉举报平台

## 项目简介

二维码投诉举报平台是一个便捷的在线投诉处理系统，通过二维码扫码方式，让群众能够快速提交投诉举报信息，工作人员可以高效处理和反馈，实现投诉处理全流程的透明化管理。

### 主要功能

1. **群众投诉界面**：扫码进入、填写投诉表单、上传证据、提交投诉
2. **处理反馈界面**：投诉状态查询、处理进度跟踪、反馈查看
3. **工作人员后台**：投诉列表管理、投诉分配、状态更新、反馈回复
4. **数据可视化**：投诉类型分布、处理效率等数据统计分析

## 技术栈

- **前端技术**：HTML5 + CSS3 + JavaScript + Tailwind CSS v3 + Font Awesome
- **数据存储**：LocalStorage（模拟数据存储）
- **二维码生成**：qrcode-generator.js
- **图表可视化**：Chart.js
